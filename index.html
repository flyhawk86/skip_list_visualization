Simple Skip List Visualization

Insertion Rule:


When we insert a number into the skip list, we start from the left sentinel of the top layer on the canvas, and walk through the skip list to find the location for insertion on the bottom layer on the canvas. Then, we insert the number into the linked list of the bottom layer. We flip a coin to decide whether we need to insert the number into the layer above recursively.

Deletion Rule:


We first check whether the number to be deleted exists in the skip list. Then, we find the number on the bottom layer on the canvas, and remove it from the linked list of this layer. We recursively check whether the number exists in any layer above to make sure that the number is removed from all layers.

Remarks:


The feature of "Promotion Times for the Next Insertion" and the feature of "Adjust the probability of the coin flip between 0.0 and 1.0" cannot be used together.
